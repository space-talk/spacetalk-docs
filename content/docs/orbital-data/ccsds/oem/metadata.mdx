# OEM Metadata

## Purpose and scope

This page enumerates all metadata items defined for an Orbit Ephemeris Message (OEM) segment. Each metadata block applies to the immediately following ephemeris data block and optional covariance block. Metadata defines the context (object, frame, time span) for interpolation.

## Block keywords and order

- A metadata block is delimited by fixed tokens:
  - META_START
  - metadata keyword assignments in fixed order (7.4.8; [Table 5-3](#fields))
  - META_STOP
- One metadata block precedes each ephemeris block (5.2.3.3).
- Comments are allowed only immediately after META_START (7.8.9).
- Multiple segments allowed for discontinuities (e.g., maneuvers); each has its own metadata.

## Validation rules specific to metadata

- Ordering: keywords must appear in the fixed order shown above (7.4.8; [Table 5-3](#fields)).
- START_TIME ≤ first ephemeris epoch ≤ last ephemeris epoch ≤ STOP_TIME.
- USEABLE_* must lie within [START_TIME, STOP_TIME]. For consecutive segments, USEABLE windows must not overlap except at a shared endpoint (5.2.4.4).
- TIME_SYSTEM must be constant across the entire OEM (5.2.4.5). CREATION_DATE in the header is always UTC (7.5.11).
- Leap second seconds value "60" is valid in time strings (7.5.10).
- All M fields mandatory; REF_FRAME_EPOCH conditional on frame (Item 8-14 in A2.5.3).

## Edge cases and handling

- Unknown IDs: use "UNKNOWN" for OBJECT_NAME/OBJECT_ID when data cannot be disclosed ([Table 5-3](#fields) note).
- Non-natural centers: CENTER_NAME may be a spacecraft or other origin; document clearly if not gravitational and avoid propagating between centers.
- TEME frame: the Blue Book indicates TEME is for OMM/TLE; avoid TEME in OEM unless governed by an ICD.
- SCLK time systems: require an ICD that defines correlation to UTC, partition handling, and epoch (Annex B-B3; Annex I).
- Max one COMMENT block per META_START.

### Multi-segment metadata:

```
META_START
OBJECT_NAME = MARS GLOBAL SURVEYOR
OBJECT_ID = 1996-062A
CENTER_NAME = MARS BARYCENTER
REF_FRAME = EME2000
TIME_SYSTEM = UTC
START_TIME = 2019-12-28T21:29:07.267
STOP_TIME = 2019-12-30T01:28:02.267
INTERPOLATION = HERMITE
INTERPOLATION_DEGREE = 7
META_STOP
... ephemeris ...
META_START
OBJECT_NAME = MARS GLOBAL SURVEYOR
... (updated times)
META_STOP
```
